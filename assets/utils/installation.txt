1. all steps are performed at the bash terminal within the visual studio.
  2. execute cli command: mkdir ~/bootcamp/homework/professionalReadmeGenerator.
  3. execute cli command: mkdir ~/bootcamp/homework/professionalReadmeGenerator/assets.
  4. within the <assets> directory, create </images> and </utils> directory.
  5. execute cli command: cd ~/bootcamp/homework/professionalReadmeGenerator
  6. execute cli command:  npm init
  7. execute cli command:  npm i inquirer@8.2.4
  8. copy index.js into the /professionalReadmeGenerator directory.
  9. copy generateMarkdown.js into the /professionalReadmeGenerator/assets/utils directory.
  10. invoke the application using the following command: node index.js.
